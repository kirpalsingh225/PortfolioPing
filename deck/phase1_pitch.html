Core Product Journey2 / 4
Technology

Built as a queue-first assistant with memory, broker context, and guardrails.

Phase 1 stack
WhatsAppMeta Cloud API for inbound and outbound messages.
FastAPIWebhook verification, message parsing, and job enqueueing.
Redis + ARQBackground processing for LLM, broker, and alert jobs.
SupabasePostgres storage for users, memory, broker tokens, alerts, and audit events.
ZerodhaKite Connect for account linking, holdings, and portfolio sync.
LangChain + Ollamagpt-oss:120b-cloud for intent, chat, and summaries.

Guardrail design

No direct advice

The assistant avoids personalized buy/sell recommendations.

Backend truth

Portfolio facts come from Supabase and Zerodha, not memory guesses.

Paper trading

Buy/sell flows require confirmation and do not place real orders.

Context memory

Recent messages plus long summaries keep conversations coherent.
